Welcome!

This documentation purpose is to show readers information on how to center align a div element with CSS.

Block Display

To center a div with a block display horizontally, what you need to do is use the following code:

  • margin-left: auto; and
  • margin-right: auto;

To do it vertically use this code instead:

  • margin-top: auto; and
  • margin-bottom: auto;

To center the div both horizontally and vertically, you just need to do one simple code which is margin: auto;

To center a div inside a div container, you can do the same step as shown before.

Flex Display

To center a single div horizontally or vertically within a flex container, you can do the following:

  • Horizontally: use justify-content: center;
  • Vertically: use align-content: center;

To align a single div both horizontally and vertically (in the middle of the div box) use both code as shown before.

For center aligning multiple div, you can see more information from this page.

Grid Display

To center a div inside a grid is a bit complicated, therefore I reccommend you to visit this page instead.

Absolute Position

To center a div using absolute position you need to do the following:

position: absolute;

top: 50%;

left: 50%;

transform: translate(-50%, -50%)

If you somehow want to align it only horizontally or vertically, you just need to change the value to suit your needs

Relative Position

To center a div with position: relative; you can use the isntruction in the Position - Absolute section. You can change the top and left into margin-top and margin-left if it doesn't work.

Fixed Position

To center a div inside a fixed parent, the easy way is to use flex display in the parent element. After that you just need to do the method as shown before in the Display - Flex section.
